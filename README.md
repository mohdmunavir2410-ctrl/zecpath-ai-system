# Zecpath AI Hiring System
This project is an AI-driven recruitment system designed to parse resumes and match candidates to job requirements.

## Project Structure
- `data/`: Contains input resumes (PDF).
- `parsers/`: Logic for extracting text from documents.
- `ats_engine/`: Keyword matching and candidate scoring.
- `utils/`: Logging and helper functions.
- `logs/`: History of system activities.

## Setup
1. Activate virtual environment: `.\venv\Scripts\activate`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the system: `python main.py`
