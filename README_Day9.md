Day 9 – Skill Extraction Engine

Objective:
To build a rule-based NLP Skill Extraction Engine.

Features Implemented:
- Master skill dictionary categorized into technical, business, creative and soft skills
- Synonym handling (ML → Machine Learning, JS → JavaScript)
- Skill stack expansion (MERN → MongoDB, Express, React, NodeJS)
- Confidence scoring based on frequency
- Duplicate removal
- Structured JSON output generation

Output:
The system extracts skills from resume text and returns categorized skills with confidence scores.
